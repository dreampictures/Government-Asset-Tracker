Analyze the structure and functionality of the website FreeJobAlert.com and build a similar job portal system.

IMPORTANT RULES

Do NOT copy the UI or design of FreeJobAlert.
Create a completely new modern UI design.

However, the system logic, content structure, and job portal workflow should work similarly to FreeJobAlert.

The goal is to build a modern automated government job portal that functions like FreeJobAlert but with a unique interface.

TECH STACK

Backend: Python Flask
Frontend: HTML, CSS, JavaScript
Database: PostgreSQL
Deployment target: Fly.io

CORE FUNCTIONALITY

The system should work as a government job notification portal that collects job notifications and publishes structured job pages.

MAIN WEBSITE STRUCTURE

Homepage sections:

Latest Jobs
Trending Jobs
Popular Jobs
Admit Cards
Results
Answer Keys

Each section should automatically display data from the database.

Sections required:

Introduction

Eligibility

AUTO JOB SCRAPER

Create a scraper module using:

requests
BeautifulSoup

The scraper should collect job information from government websites and store it in the database.

Run the scraper automatically once per day.

AI CONTENT GENERATION

Use free open-source AI models with Ollama and Llama3 to convert job data into structured SEO articles.

The article should automatically fill sections such as introduction, eligibility, vacancy details, and application process.

TELEGRAM JOB ALERT SYSTEM

Create a Telegram bot that automatically posts job alerts when new jobs are added.

EMAIL ALERT SYSTEM

Allow users to subscribe to job alerts and receive notifications when new jobs are published.

Optimize for:

UI REQUIREMENTS

Design a modern interface using card layouts and clean typography.

The UI must be unique and not similar to FreeJobAlert design.

Focus on usability, readability, and mobile experience.

FINAL GOAL

Build a modern automated job portal system that functions like FreeJobAlert but uses a completely unique design and modern interface.
